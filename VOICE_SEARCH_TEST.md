## 🎤 语音输入功能

### 功能说明
- 使用浏览器内置 Web Speech API 进行语音识别
- 支持中文语音输入
- 自动将语音转为文字并触发 AI 解析

### 使用方法
1. 在首页"快速记账"区域，选择"文字输入"模式
2. 点击输入框右侧的 🎤 麦克风图标
3. 开始说话（例如："午餐吃了35块钱的牛肉面"）
4. 语音识别完成后自动填充到输入框
5. 自动触发 AI 解析

### 浏览器支持
- ✅ Chrome / Edge（推荐）
- ✅ Safari（iOS/macOS）
- ❌ Firefox（不支持 Web Speech API）

### 注意事项
- 首次使用需要授权麦克风权限
- 需要在安静环境下使用
- 识别准确率取决于发音清晰度

## 🔍 搜索筛选功能

### 功能说明
- 在"记录"页面顶部添加搜索筛选栏
- 支持多种筛选条件的组合使用

### 筛选条件

#### 1. 关键词搜索
- 搜索详情、商户名称
- 实时搜索，支持中文

#### 2. 类别筛选
- 全部类别
- 餐饮 🍔、交通 🚌、购物 🛒、居住 🏠
- 医疗 💊、娱乐 🎬、收入 💰、其他 📦

#### 3. 日期筛选
快捷预设：
- 今天
- 最近7天
- 最近30天
- 本月
- 上月

自定义日期范围：
- 开始日期
- 结束日期

#### 4. 金额筛选（高级筛选）
- 最小金额
- 最大金额

### 使用方法
1. 切换到"记录"标签页
2. 在搜索框输入关键词，或选择筛选条件
3. 点击"搜索"按钮
4. 查看筛选结果和统计信息
5. 点击"重置所有筛选"恢复默认列表
