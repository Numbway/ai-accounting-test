## 📋 项目概述

基于 AI 的智能记账软件，支持自然语言输入和图片 OCR 识别，自动解析消费信息并记录。

### 技术栈
- **后端**: FastAPI + SQLAlchemy + SQLite
- **前端**: React + Vite + Tailwind CSS + Recharts
- **AI**: 阿里云 qwen / OpenAI / DeepSeek API

## 🚀 开发计划（按优先级排序）

### Phase 1: 核心功能完善 🔥

#### 1.1 编辑/删除记录
- **优先级**: P0
- **工作量**: 1-2 小时
- **功能描述**:
  - 前端：在支出列表中添加编辑和删除按钮
  - 后端：PUT /api/expenses/{id} 和 DELETE /api/expenses/{id} 已存在，只需对接
  - 编辑弹窗：修改金额、类别、日期、备注
  - 删除确认：二次确认对话框
- **验收标准**:
  - 可以修改任意字段
  - 删除后列表自动刷新
  - 操作有成功/失败提示

#### 1.2 图片 OCR 识别
- **优先级**: P0
- **工作量**: 2-3 小时
- **功能描述**:
  - 支持上传购物小票图片
  - OCR 提取文字（阿里云/百度/腾讯 OCR）
  - AI 解析提取金额、商户、日期、商品
  - 预览识别结果，支持手动修正
- **技术方案**:
  - 方案 A: 阿里云 OCR（推荐，国内速度快）
  - 方案 B: 百度 OCR（免费额度高）
- **验收标准**:
  - 支持 JPG/PNG 格式
  - 识别准确率 > 80%
  - 支持手动修正识别结果

#### 1.3 数据导出
- **优先级**: P1
- **工作量**: 1 小时
- **功能描述**:
  - 导出为 CSV 格式
  - 支持按时间范围筛选
  - 后端已提供 /api/export/csv，只需前端对接
- **验收标准**:
  - 可以下载 .csv 文件
  - 包含所有字段
  - 中文不乱码

### Phase 2: 增强功能 ⭐

#### 2.1 预算设置
- **优先级**: P1
- **工作量**: 2 小时
- **功能描述**:
  - 设置月度预算上限
  - 实时显示预算使用率
  - 超支提醒（进度条变红）
- **数据库变更**:
  - 新增 `budgets` 表：month, year, amount
- **验收标准**:
  - 首页显示预算进度条
  - 超支时明显提示

#### 2.2 语音输入
- **优先级**: P2
- **工作量**: 2-3 小时
- **功能描述**:
  - 长按麦克风按钮录音
  - 语音识别转文字
  - 调用现有 AI 解析接口
- **技术方案**:
  - Web Speech API（浏览器内置）
  - 或阿里云语音识别
- **验收标准**:
  - 支持中文语音
  - 识别准确率可接受

#### 2.3 搜索和筛选
- **优先级**: P2
- **工作量**: 1-2 小时
- **功能描述**:
  - 按关键词搜索（详情、商户）
  - 按类别筛选
  - 按日期范围筛选
- **验收标准**:
  - 实时搜索，无需刷新
  - 支持组合筛选条件

---

### Phase 3: 高级功能 🌟

#### 3.1 多用户支持
- **优先级**: P3
- **工作量**: 4-6 小时
- **功能描述**:
  - 用户注册/登录
  - JWT Token 认证
  - 数据隔离
- **数据库变更**:
  - 新增 `users` 表
  - `expenses` 表添加 user_id 外键
- **验收标准**:
  - 只能看到自己的数据
  - Token 过期自动跳转登录

#### 3.2 数据同步/备份
- **优先级**: P3
- **工作量**: 4-6 小时
- **功能描述**:
  - 云端数据备份
  - 多设备同步
  - 数据恢复
- **技术方案**:
  - 后端部署到云服务器
  - 或支持导出/导入备份文件

#### 3.3 智能分析
- **优先级**: P3
- **工作量**: 3-4 小时
- **功能描述**:
  - 消费趋势分析（折线图）
  - 异常消费提醒
  - 节省建议（AI 生成）
